======================================
🧾 CHECKGENERATOR! — Receipt Generator
======================================

by Richard Moiseiev '@the_industrialist'

🟢 HOW TO START:
/*++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++*/
1. Download the `checkGenerator!` folder to your PC.
2. Run `checkGenerator!.exe`
3. Enjoy automatic receipt generation with a QR code and ASCII cats 🐱

📦 No installation required. It works right out of the box.
/*++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++*/

🛠️ USAGE:
/*++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++*/
Add or import items list:

1. To add items manually:
   - Enter product name, quantity, and unit price.
   - Click "Add" — the item will appear in the list.

2. To load items from CSV or TXT:
   - Click "Load from File".
   - Select a `.csv` or `.txt` file in the format:
     
     Name,Quantity,Price,Total
     Example: Milk,2,35.00,70.00
     

3. Generate PDF receipt:
   - Click "Save as PDF".
   - Choose a file name and save location.
   - The PDF will include:
     - Store name
     - Item list table
     - Total & change
     - A cat 🐾
     - A QR code at the bottom

4. Clear item list:
   - Click "Clear" to start over.
